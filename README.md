## 📱 About FitGenius

FitGenius is an all-in-one fitness app combining:
- 📋 Personalized Training Plans
- 🍎 AI-Powered Diet Analysis
- 📊 Progress Statistics
- 📱 iOS Widgets

## 📄 Privacy Policy

This repository contains the privacy policy for FitGenius, which outlines:

- **Data Collection**: We don't collect any personal information
- **Local Storage**: All user data is stored locally on device
- **iCloud Sync**: If enabled, data syncs through user's personal iCloud account
- **AI Analysis**: Food photos are processed via Aliyun Qwen API only when user actively uses the feature
- **Apple Sign In**: Authentication handled entirely by Apple
